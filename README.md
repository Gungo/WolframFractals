# WolframFractals
A python implementation of Wolfram's rule_30 binary map.

Wolfram's 2D binary fractals follow one of 128 rules. Rule 30 happens to be the most impressive.

![Output Image](https://i.imgur.com/YheDbhK.png)

## Run: 
`python rule_thirty.py`
